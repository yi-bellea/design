## 4. 간격 & 그리드

### 그리드 시스템

#### PC
```
최대 너비:     1200px
컬럼:          12 column
거터(Gutter):  20px
마진(Margin):  좌우 각 24px
```

#### MO
```
기준 너비:     320px
컬럼:          4 column
거터(Gutter):  12px
마진(Margin):  좌우 각 16px
```

### 간격 단위 (Spacing Scale)

| 토큰 | 값 | 사용 예 |
|------|----|---------|
| `space-4` | 4px | 아이콘 ↔ 텍스트 |
| `space-8` | 8px | 태그 내부 패딩, 소 여백 |
| `space-12` | 12px | 컴포넌트 내부 기본 패딩 |
| `space-16` | 16px | 카드 내부 패딩, 리스트 항목 간격 |
| `space-20` | 20px | 섹션 간 소 여백 |
| `space-24` | 24px | 섹션 내부 패딩 |
| `space-32` | 32px | 섹션 간 중간 여백 |
| `space-40` | 40px | 섹션 간 대 여백 |
| `space-48` | 48px | 페이지 상하 여백 |

### 컴포넌트 패딩 기준

| 컴포넌트 | PADDING (상/하/좌/우) |
|----------|-----------------------|
| 카드 (PC) | 24px / 24px / 24px / 24px |
| 카드 (MO) | 16px / 16px / 16px / 16px |
| 버튼 (대) | 14px / 14px / 24px / 24px |
| 버튼 (소) | 8px / 8px / 16px / 16px |
| 태그/배지 | 4px / 4px / 8px / 8px |

## 5. 공통 컴포넌트

### 5-1. 버튼

#### 종류

| 이름 | 설명 |
|------|------|
| `btn_primary` | 주요 CTA — `#0064D2` 배경, 흰 텍스트 |
| `btn_secondary` | 보조 버튼 — 흰 배경, `#0064D2` 테두리 |
| `btn_ghost` | 텍스트 버튼 — 테두리 없음 |
| `btn_carList_util` | 차량 리스트 유틸리티 버튼 (찜, 비교 등) |
| `PC_btn_search` | 검색 실행 버튼 |
| `하단 플로팅` | 모바일 하단 고정 CTA 버튼 |

#### 상태
- `default` / `hover` / `active` / `disabled` / `loading`

---

### 5-2. 상품 카드

#### PC 카드 (`PC상품-기본`)
```
너비:      240px (4열 그리드 기준)
이미지:    240 × 180px (3:2 비율)
내부 패딩: 12px
```

| 요소 | 스펙 |
|------|------|
| 차량 썸네일 | 상단 이미지 영역, 좌상단에 상태 태그 표시 |
| 상품 구분 태그 | `tag_상품구분` — 인증/일반 구분 |
| 홈서비스 태그 | `tag_홈서비스` — 오렌지 배지 |
| 차량명 | 14px / 600 / `#1A1A1A` (2줄까지 허용) |
| 연식·주행거리 | 12px / 400 / `#666666` |
| 가격 | 16px / 700 / `#1A1A1A` |
| 할부 정보 | 12px / 400 / `#1A6FFF` |

#### MO 카드 (`MO상품-기본`)
```
너비:      320px (풀 폭)
이미지:    320 × 220px
```

## 6. 페이지 템플릿

### 6-1. 내차사기 상품 상세 (PC) — `PC_01내차사기_상세_v2_0`

```
레이아웃 구성:
┌─────────────────────────────────────────┐
│ GNB (로고 + 주요 메뉴 + 로그인)         │
├─────────────────────────────────────────┤
│ 이미지 갤러리 (메인 1 + 서브 4)         │
├──────────────────────┬──────────────────┤
│ 차량 기본 정보       │ 가격 + CTA       │
│ (제목, 연식, 주행)   │ (구매하기 버튼)  │
├──────────────────────┴──────────────────┤
│ 편의 옵션 (가로 스크롤)                 │
├─────────────────────────────────────────┤
│ AI 추천 가격 분석 박스                  │
├─────────────────────────────────────────┤
│ 차량 성능 및 관리 상태                  │
├─────────────────────────────────────────┤
│ 세스코 카카어 살균/탈취 완료 배너       │
├─────────────────────────────────────────┤
│ 추천하는 이유 (담당자 프로필)           │
├─────────────────────────────────────────┤
│ 만족 구매 후기                          │
├─────────────────────────────────────────┤
│ AW플러스 보증 배너                      │
├─────────────────────────────────────────┤
│ 이제 함께서 한나오 홈서비스             │
├─────────────────────────────────────────┤
│ 차량 위치 (지도)                        │
├─────────────────────────────────────────┤
│ 자주 묻는 질문 (FAQ)                   │
├─────────────────────────────────────────┤
│ 동급 추천 차량 (가로 스크롤 카드)       │
├─────────────────────────────────────────┤
│ Footer                                  │
└─────────────────────────────────────────┘
```

| 차량 기본 정보 | `#FFFFFF` | 상하 32px |
